## 博客
> 使用vue开发的博客

### 为什么要重新开发自己的博客
> 由于学习过一段时间的vuejs,对这种MVVM数据视图双向绑定的模式非常感兴趣,由此第一个想到的就是拿自己以前写的博客页做实践。

### 设计到的技术
> 前端
```
'vue' + 'vue-router' +'vue-resource'+bootstrap+vue衍生的组件库Element+webpack
```
> 服务端[nodejs]
```
express+mongoose+bluebird+七牛存储
```
